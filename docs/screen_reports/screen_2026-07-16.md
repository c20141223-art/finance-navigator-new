## 覆算方式

每格因子以「原始值 → 得分」呈現。對照文末 momentum.yaml：
bins 由上而下第一個命中給分（`gt` 嚴格大於、`gte` 含等於、
`range: [a, b]` 為 a ≤ 值 < b）；每維度兩因子得分相加、超過 100 以
100 計；總分 = 0.30×基本面 + 0.35×籌碼 + 0.35×技術。
原始值的窗口定義見 `stock_screener/momentum.py` 模組 docstring
（60MA 用還原收盤價；量以張計、口徑為官方日成交資料）。

> **同分檔位排序無鑑別意義**：排名以總分遞減排序，同分時僅以
> 證券代號升冪做確定性 tie-break（讓同一天重跑結果可重現），
> 代號順序不含任何資訊。目前 bins 較粗、維度雙因子封頂容易達標，
> 前段出現同分群為現行參數的特性而非 bug；依回饋機制紀律，參數
> 調整需 ≥ 30 個觸發案例的證據支撐，故不預先調整。

> **已知資料限制一（上櫃籌碼）**：TPEx 三大法人 openapi 無歷史端
> 點，剛回補完的資料庫中上櫃個股僅有最新一日法人資料，其「投信連
> 買天數」與「外資佔比」原始值忠實反映資料庫現況（連買最多 1、佔
> 比僅含單日），會隨每日排程累積而正常化。上市個股（T86 有歷史）
> 不受影響。
>
> **已知資料限制二（營收趨勢）**：月營收來源為快照端點、無歷史可
> 回補（已查證兩交易所 openapi 目錄內所有端點皆不帶參數），資料庫
> 每月僅累積一個月份。`revenue_trend_3m` 需連續 3 個月資料，系統
> 上線後約 3 個月才開始有意義；在此之前恆為 ✗→0，基本面分項實質
> 上只剩 revenue_yoy 單因子（滿分上限仍為 100，不受影響）。

# 順勢評分 profile（規格書第 2、3 節）。所有門檻與權重集中於此，
# 程式碼不含硬編碼參數。調整紀律見 README「版本紀律」一節。
#
# bins 語意（stock_screener/scoring.py 的 score_bins）：由上而下逐條比對，
# 第一個命中的給分；{gt: x} 表示 值 > x、{gte: x} 表示 值 >= x、
# {range: [a, b]} 表示 a <= 值 < b（含下界、不含上界）、{else: s} 為
# 保底。原始值缺漏（None）一律 0 分（規格書 1.4：資料不足的因子計 0 分）。

filters:
  # 近 5 個交易日均量 > 500 張（成交量口徑：TWSE/TPEx 官方日成交資料，
  # 見 docs/api_samples/README.md）
  min_avg_volume_5d: 500
  # 收盤價（原始市場價，非還原價）> 10 元
  min_price: 10
  # 具備至少 60 個交易日歷史（同時實作規格書 1.4 的新上市股排除：
  # 在最近 min_history_days 個全市場交易日中，該股必須每日都有資料）
  min_history_days: 60
  # 只納入普通股（4 碼數字且非 0 開頭的代號；排除 ETF、權證、債券等）
  common_stock_only: true

dimensions:
  fundamental:
    weight: 0.30
    factors:
      # F1 月營收動能：最新可得月份的單月營收年增率（%）
      revenue_yoy:
        bins:
          - {gt: 30, score: 100}
          - {gt: 10, score: 60}
          - {gt: 0, score: 30}
          - {else: 0}
      # F2 營收趨勢：最近 3 個「連續」月份的 YoY 逐月走高（Boolean）。
      # 注意：月營收來源為快照端點、無歷史可回補（兩交易所 openapi 目錄
      # 皆無帶參數端點），此因子需系統上線每日排程累積約 3 個月才開始
      # 有意義；在此之前恆為 0，基本面分項實質上只剩 revenue_yoy 單因子。
      revenue_trend_3m:
        score_if_true: 100

  chips:
    weight: 0.35
    factors:
      # C1 投信連買天數：自篩選日往前連續 trust_net > 0 的交易日數
      #（缺該日資料視為中斷）
      trust_buy_streak:
        bins:
          - {gte: 10, score: 100}
          - {gte: 5, score: 70}
          - {gte: 3, score: 40}
          - {else: 0}
      # C2 外資買超佔比：近 5 個交易日外資買賣超合計（張）÷ 同期間
      # 該股總成交量（張）
      foreign_net_ratio_5d:
        bins:
          - {gt: 0.15, score: 100}
          - {gt: 0.05, score: 60}
          - {else: 0}

  technical:
    weight: 0.35
    factors:
      # T1 趨勢位階：還原收盤價相對還原價 60MA 的乖離率（%）。
      # 剛站上（0~8%）高分、8~20% 遞減、20~25% 低分、<0% 或 >25% 零分
